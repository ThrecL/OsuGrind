# OsuGrind

OsuGrind is a professional performance analytics and habit-building companion for [osu!](https://osu.ppy.sh/). It provides high-precision, real-time stat tracking and deep post-play analysis to help you understand your timing, aim, and consistency bottlenecks.

Visit the official website at [osugrind.app](https://osugrind.app).
